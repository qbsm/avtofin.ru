##Сборка проекта

###Требования к окружению
npm
gulp
webpack
composer

###Файлы конфигурации
`gulpfile.babel.js` конфиг gulp
`webpack.config.js` конфиг webpack

###Инициализация проекта
`npm run init` - установятся все зависимости

###Разработка CSS, Twig, Img
`npm run watch`

###Разработка JS
`npm run watch-js`

###Production-cборка
`npm run build`
